🎬 MovieHub – Watch, Download & Buy Movies

🌟 Overview
MovieHub is an online movie streaming and purchasing platform where users can watch, download, and buy their favorite movies. It offers a premium movie collection, high-speed downloads, and a secure payment system for purchasing movies.

🚀 Live Demo
🔗 [Live Project Link](https://movie-hub-client.web.app/)

🛠️ Technologies Used
Frontend	Backend	Database	Authentication	Payments	Hosting
React.js, Next.js	Node.js, Express.js	MongoDB / PostgreSQL	JWT / Firebase	Stripe / PayPal	Vercel / AWS / DigitalOcean
📌 Core Features
✅ User Dashboard

Browse & watch movies online
Download movies in different resolutions
Buy premium movies securely
✅ Admin Panel

Manage movie uploads & categories
Track purchases & user activities
Handle refund & withdrawal requests
✅ Payment System

Integrated Stripe/PayPal for secure transactions
Subscription & one-time purchase options
✅ Additional Features

Search & filter movies by genre, rating, year
User reviews & ratings
Multi-device compatibility (Mobile, TV, Desktop)
